# School Management System

## Overview
A Java desktop application designed to manage students, teachers, attendance, and
assignments in a school environment.

## Features
- Student and teacher management
- Attendance tracking
- Assignment management
- Database-backed persistence using SQLite

## Technologies Used
- Java
- JavaFX
- SQLite
- JDBC

## Architecture
- Model-View-Controller (MVC) design pattern
- Data Access Object (DAO) pattern for database operations
